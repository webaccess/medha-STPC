# Installation
## Frontend
- Technology used: ReactJS
- Steps for project setup: 
  - Create a file .env
  - Copy contents of file .env.txt to file .env
  - In terminal use below commands:
    - cd webapp
    - npm install
## Backend
- Technology used: Strapi.io
- Database: PostgreSql
